## 执行步骤

1. 从四个夹获取 BVID 列表 → `cache/reclassify-pending.json`
2. 用 resource/list 获取每夹的视频标题（速率：每夹 1次 API）
3. 批量发送 DeepSeek，提示词：

```
你是B站收藏夹分类助手。以下视频标题需要归入最合适的文件夹。

可用文件夹: [文件夹列表]

对每个视频，输出 JSON:
{"bvid": "...", "suggested_folder": "...", "reason": "..."}

规则:
- AC/动漫番剧/游戏 三个夹已合并，不要分配
- 电影解说 → 电影
- 鬼畜独立，不放搞笑
- 无法判断的放回原夹
```

4. 解析 LLM 输出 → 按新文件夹分组 → 转 avid → 批量 move
5. 验证
